음료 제조 공정 MES 시스템

2025.10.02 ~ 2025.10.28 (4주)
Spring MVC + Oracle + MyBatis
DB 설계(ERD), 생산/불량 프로세스 로직 구현, AES256 암호화 모듈 개발

System Architecture & Design

Oracle 10g를 기반으로 제3정규화까지 적용된 ERD 설계를 통해 데이터 중복을 최소화했습니다. Use Case 설계를 통해 관리자(Admin)와 작업자(Worker)의 권한을 분리하여 보안성을 강화했습니다.

Key Features

실시간 통합 대시보드: Chart.js를 활용하여 당일 생산량(1,613건)과 불량률(3.2%)을 실시간으로 집계하고 시각화했습니다.

생산 및 재고 현황: 라인 차트와 파이 차트를 통해 주간 생산 추이와 불량 유형 분포를 한눈에 파악할 수 있습니다.

Process & Logic

불량 승인 프로세스: 작업자가 등록한 불량 내역을 관리자가 검토 후 승인(APPROVED) 처리하는 트랜잭션 로직을 구현했습니다.

사용자 권한 관리: AES256 암호화를 적용하여 개인정보를 보호하고, 관리자/작업자 권한에 따라 메뉴 접근을 제어합니다.
